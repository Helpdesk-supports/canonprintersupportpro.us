Required Methods To Connect MacBook Air To Canon Wireless Printer

Here are the various ways through which one can effortlessly connect the devices. So, keep your eyes feast on the below-mentioned steps:

Method 1: If Your Canon Printer Has A Built-in Display

Printers with a touchscreen or other built-in display (control panel) generally permit you to use that display to choose the wi-fi network and enter the name and its password. Check the printer’s documentation for details.

Method 2: If Your Canon Printer Uses WPS (Wi-Fi Protected Setup

If your )Wi-Fi router isn’t created by Apple, check the router’s documentation for details about how to add a WPS printer. If your Wi-Fi router is an AirPort Base Station, follow the given steps:

  • First, go to the “Applications” folder, select “AirPort Utility” under the “Utilities folder” section
  • Now, choose your base station in AirPort Utility, then enter the base station password, if required
  • From the menu bar, choose Base Station > Add WPS Printer
  • Select either “First attempt” or “PIN” to start making connection. After that, click “Continue” tab
  • If you go with the option “First attempt,” push the WPS button on the printer. When the printer’s MAC Address shows in AirPort Utility, click “Done”
  • If you go for “PIN,” enter the printer’s PIN number, which should be listed in the printer’s documentation and then press “Continue”. When the printer’s MAC Address appears in AirPort Utility, click “Done”
  • At the end close AirPort Utility

Method 3: If Your Canon Printer Can Connect To Your Mac Via USB Cable

You can use a USB connection to set up Wi-Fi printing. Follow the below points:

  • Connect your Canon printer to your MacBook with the use of an appropriate USB cable
  • Install the Mac software that came with the printer, and make sure that it includes a printer setup assistant
  • Use the printer setup assistant to connect the printer to your Wi-Fi network. Check the printer’s documentation for details
  • Disconnect the USB cable from the printer and Mac. The printer will yet be connected to the Wi-Fi network
