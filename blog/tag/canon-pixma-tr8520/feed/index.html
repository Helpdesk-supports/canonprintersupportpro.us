So let us start the guide to remove the offline error.

Bring Your Canon Pixma Tr8520 Printer Online

Follow the below simple steps to change the status of your Canon printer from offline to online. First, you need to check a few things before starting with troubleshooting steps:

  1. Check if the printer is ON or should be not in the sleep mode.
  2. Check to ensure that both the ends of the USB cable are firmly inserted.
  3. Make sure that the printer is getting a complete network.
  4. Ensure that you have to download the right printer drivers and they are updated.
  5. If you have installed new Windows, then make sure to update the drivers according to it.
  6. Check if there is any Paper-jam issue.

Now let’s see troubleshooting steps

Step 1: Remove Printing Jobs

  1. Click the Start button or Windows icon
  2. Now choose Settings
  3. Then select Devices in the Settings window
  4. Click Printers and Scanners
  5. Now select your printer and right-click on it.
  6. In the drop-down menu open “See what’s printing”
  7. Then select all the previous printing jobs and delete them.

Step 2: Change Printer Status

  1. Open Printers and Scanners window
  2. Again select your printer and right-click
  3. This time select Printer and choose- “Use printer online” option.

This will surely make your printer come online and then you can again try getting prints from the Canon printer. However, it still shows the printer offline issue then uninstall the printer drivers.

Step 3: Reinstall The Printer Drivers

  1. To uninstall the printer driver open Printer Server Properties window
  2. Now remove the printer driver
  3. Once it gets uninstall completely
  4. Now open Printers and Scanners window again and add printer
  5. Then install the Printer drivers for your printer.

This step is the last option to solve the printer errors, and while solving the Canon Pixma tr8520 printer offline issue you can use this method to again make your printer online. While turning your Canon printer offline to online, you may face issues.
