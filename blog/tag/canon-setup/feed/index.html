Effective Steps to Connect Canon iP8720 Printer to Wi-Fi

Here are the step-by-step guidelines which will assist you to make your Canon iP8720 printer connected with Wi-Fi network. If you are wandering for the obtaining the same type of answer, then stick your eyes to the below-noted points:

Few Things You Should Note About The Network Connection

  • Security settings for the network devices, setup instructions, router functions, and configurations may be different according to the environment where the system is put in.
  • Take a brief glimpse at the network administrator if this is an Office setup
  • Ensure that the network you are accessing on is completely secured. Otherwise, third party may try to access your personal information.
  • Your usable printer and computer system is working on the same network

Install Printer Driver And Software

For using the printer which is connected to your operating system, having an installed driver on PC is must needed. Therefore, download and install the actual driver for your Canon iP8720 Printer.

Important Notes You Should Keep In Mind

  • First, log in as an administrator to connect your canon ip8720 printer to Wi-Fi
  • Close all the apps you accessed before setup
  • Don’t ever try to change the users during the driver installation process
  • For installing software, an internet connection is must require on computer
  • After completion of software installation successfully, you need to reboot the computer

Setup Your PIXMA iP8720 With Below Noted Steps

  • Visit the site JI setup
  • Appropriately, follow the guidance that are directed in the product setup section, download now
  • Open the .dmg file

When you open this file, the setup procedure will automatically get started. Without any cable, installation method shows up on the screen. Neither you need to mention the network name in this sort of configuration, nor have to enter the password. You don’t need to use a USB wire for connecting your computer to the printer.

Now Select Connect To The Network Once The Printer Setup Properly Done

  • Choose cableless protection from the setup security selection
  • Stay holding the Wi-Fi button on your computer to connect the ip8720 printer to the Wi-Fi until the power light flashes twice in a row

  • Now, choose Canon ip8700 series with Canon IJ network. After that click on “Add”

Steps for canon MF4720W printer Wi-Fi setup

There are two methods for connecting this printer to wireless network points:

  • Configuring by using WPS (Wi-Fi Protected Setup)
  • Connecting Manually

Configure canon MF4720W with WPS Features:

If your wireless LAN routers or access points support WPS features, then you can connect canon wireless printers by using push button or PIN code system.

Push Button System:

  • In this you have to select WPS push button mode under the network settings of your printer.

PIN Code System:

  • In this you need to enter PIN code which is generated by the printer to wireless LAN router or access points for connection.

Have a look at the steps for Configuring Manually

When Wireless LAN routers or access points don’t support WPS, then you need to enter manually wireless LAN router information to the machine for connection.

By selecting Access Point:

  • First you need to search for all access points by the canon MF4720W printer.
  • Select a wireless LAN router or access point which is to  connected and then enter a network key for connection.

By entering manually:

  • You have to enter manually SSID or security information of wireless LAN routers or access points that is to be connected to the canon MF4720W printer.

 Here are the steps for MF4700 Wireless connection setup:

Connecting canon MF4700 to a wireless network is so simple you just need to connect it to WPS physically.

Look at the requirements for establishing connection setup:

  • Should have push-button WPS physically
  • There should be WPA or WPA2 security

Follow the below mentioned steps in the given order to connect it swiftly:

  • Press the home printer button.
  • By using navigation keys/arrows select WLAN setup icon.
  • Go for left function
  • Press the left function it will turn on the Wi-Fi light
  • Now, the printer will search for access that supports WPS.
  • Keep pressing and holding WPS button for at least 5 seconds
  • After 2 minutes press the OK button
  • When the connection is complete press OK to finish the configuration.
